## Description

nest microservices with RabbitMQ, Redis and Postgres

## Installation

```bash
$ yarn install or npm install
```

## Running the app

```bash
# Inicia serviços em docker
$ sudo docker compose up -d
```

```bash
# Finaliza serviços em docker
$ sudo docker compose down
```
